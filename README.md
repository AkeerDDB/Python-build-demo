This is a simple Python application that calculates the total of a list of values.  
The project demonstrates:

- basic Python functionality (`calculate_total`)
- automated testing with pytest
- Continuous Integration (CI) using GitHub Actions
- code quality checks with flake8

Run tests locally:

```bash
pytest
```
